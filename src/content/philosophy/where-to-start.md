哲学書じゃなく、**手を動かす**。

オススメの第一歩:
1. Copilot CLI をインストール
2. 自分の日常で**3回以上やった作業**を1つ選ぶ
3. それを skill にする（`SKILL.md` を書くだけ）
4. 翌週、また使ってみる

完璧じゃなくていい。**動くものを今日中に**。
小さな勝ちが積み上がって、ある日気付く——「AI なしじゃ仕事できない」と。
