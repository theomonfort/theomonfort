エンジニアの第二の故郷、それがターミナル。Copilot CLI はそこに住むエージェントだ。

**できること:**
- シェルのコマンドを提案・説明する
- リポジトリを読んで PR を作る
- スキルや MCP server を読み込んで拡張する
- バックグラウンドで長時間タスクを回す

**ハッカーの遊び方:**
```bash
copilot           # 対話モード
copilot -p "このリポジトリの主要モジュールを3行で"  # 一発実行
```

CLI に慣れると、君の手元の生産性が**指数関数的に**伸びる。
