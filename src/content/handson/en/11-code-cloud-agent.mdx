### 5.1 — Assign an existing Issue to the Cloud Agent

Pick one of the Issues you created in Step 6.5 and let Cloud Agent implement it.

1. On github.com, open your repository.
2. Click the **Issues** tab at the top.
3. From the list, open one of the Issues you want Cloud Agent to work on.
4. In the right sidebar, click **Assignees**.
5. Pick one of **Copilot** (GitHub), **Claude** (Anthropic), or **Codex** (OpenAI) from the list and assign it.

![Cloud Agent assignment screen](/theomonfort/handson/img/cloud-agent-assignees.png)

When you assign, you can customize:

- **Additional prompt** — extra instructions added on top of the Issue description.
- **Model selection** — pick from Copilot / Claude / Codex.
- **Base branch** — the branch the work starts from.

<Aside type="positive">

**Key point**: The Cloud Agent will implement the code autonomously and open a PR (⏱ ~15 min per Issue). If the Validation Tools you set up in Step 4.1 (CodeQL, Code Review, Secret Scanning) are enabled, the Agent will verify its own implementation before submitting the PR. You can assign multiple Issues at once and they'll be worked on in parallel.

</Aside>

<Aside type="reference">

**Leave a few Issues for the Part 13 CLI session**: in Part 13 we'll use Copilot CLI's `/fleet` to implement the remaining open Issues in parallel. **Don't assign every Issue from Step 6.5 here — leave 2–3 of them unassigned** so the CLI has something to chew on later.

</Aside>

### 5.2 — Review and merge the PR

The PR the Cloud Agent creates is opened as a **Draft**.

1. Open the PR and review the changes.
2. Click **Ready for review** to lift the draft state.
3. Copilot Code Review kicks off automatically.
4. Once the review is done, merge the PR.

<Aside type="negative">

**If you hit a conflict**: when multiple Cloud Agents are working in parallel, they can touch the same files and cause merge conflicts. Click **Resolve conflicts** → **Fix with Copilot** and Copilot will resolve the conflict for you automatically.

</Aside>

![Resolving conflicts with Fix with Copilot](/theomonfort/handson/img/fix-conflicts-copilot.png)

### 5.3 — Pull the latest code and try it out

Once the Cloud Agent's PR is merged, pull the latest code into your Codespace and check the site.

```bash
git checkout main && git pull && npm install && npm run dev
```

When the port forwarding notification pops up, click **"Open in Browser"** to try out the new feature the Cloud Agent shipped.
