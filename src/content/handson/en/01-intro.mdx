In this workshop, you'll get hands-on with the simple concepts introduced in the [**Copilot Playbook**](https://theomonfort.github.io/theomonfort/en/playbook/github/?present=1). Following the **PLAN → CODE → REVIEW → TEST & SECURE → OPERATE** phases, you'll use GitHub Copilot to ultimately build a simplified version of the Playbook (a site for cleanly browsing Markdown content) together with Copilot.

### Today's goals

- **PLAN**: Equip Copilot with "context", "rules", and "shape" via MCP, Instructions, and Skills
- **CODE**: Design and implement the site using Copilot Chat (Plan / Agent)
- **REVIEW**: Automatically review PRs with Copilot Code Review
- **TEST & SECURE**: Guarantee quality and safety with Dependabot, CodeQL (GHAS), and GitHub Actions (Playwright)
- **OPERATE**: Build AI into daily operations with Cloud Agent, Copilot CLI, and Agentic Workflows

### Prerequisites

Please prepare the following environment:

- A private Organization with the following features enabled:
  - GitHub Actions
  - GitHub Copilot (Pro+ / Business / Enterprise)
  - GitHub Advanced Security (GHAS)
- Permission to create repositories within that Organization
