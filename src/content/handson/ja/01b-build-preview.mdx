## ゴール ── 「**今、あなたが見ているこのサイト**」を作ります

実はこの勉強会のゴールは、特別なデモアプリではありません。

**今あなたが読んでいる、この [Copilot Playbook](https://theomonfort.github.io/theomonfort/playbook/github/?present=1) サイトそのもの（の簡略版）** を、Copilot と一緒にゼロから作り直します。

<img src="/theomonfort/handson/img/build-preview-playbook.png" alt="今日作る Copilot Playbook サイト" class="handson-build-preview" />

### なぜこのサイトを作るのか

- **ゴールが目に見える** ── 今読んでいる画面そのものが完成形なので、毎ステップで「何のために」が明確
- **Copilot の各機能が活きる場面が分かる** ── MCP / Instructions / Skills / Chat / Code Review / Actions / Cloud Agent を、実際に必要な場面で使う
- **持ち帰れる** ── 完成したサイトはそのまま社内ドキュメント / プレイブックの土台として再利用できる

<Aside type="positive">

**💡 ポイント**: 完成像はすでに目の前にあります。ハンズオン中に迷ったら、いつでもこのサイトに戻って「次に作るのはどの部分か」を確認してください。

</Aside>
