### 9.1 — CodeQL を有効化する

1. リポジトリの **Settings** → **Security and quality** → **Advanced Security**
2. **Tools** → **CodeQL analysis** → **Set up** → **Default** → **Enable CodeQL**

<Aside type="negative">

**注意**: CodeQL はデフォルトブランチに存在する言語のみスキャンします。最初の Astro PR をマージしてから CodeQL を有効化すると、以降の Push / PR で TypeScript / JavaScript の自動スキャンが実行されます。

</Aside>

<Aside type="positive">

**ポイント**:

- **検出内容**: CodeQL は静的解析で XSS / SQL Injection などのセキュリティ脆弱性を検出します。
- **実行タイミング**: 有効化直後にデフォルトブランチ全体を初回スキャン → その後はデフォルトブランチへの push、PR、週次スケジュールで自動実行されます（PR のときだけではありません）。
- **結果の確認場所**: リポジトリの **Security** タブ → **Code scanning** から全アラートを確認できます。PR では該当行に直接コメントが付きます。

</Aside>
