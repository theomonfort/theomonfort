**Competitor News**
_This section covers select updates and analysis from known and emerging competitors._  

**AI**

* **Anthropic** [introduced an analytics admin api feature for Claude Code](https://docs.claude.com/en/api/claude-code-analytics-api). The API includes metrics on tool usage, token costs and productivity metrics. Access is limited to enterprise users. SEI vendors such as **Jellyfish** (see [Mona Monitor 34](https://github.com/github/Business-Insights/discussions/43)) will expand recently announced integrations with Claude Code using this API.

* **Anthropic** [announced](https://www.anthropic.com/news/claude-in-xcode) Claude is available in XCode 26.
 
* **OpenAI** released [GPT5-Codex](https://openai.com/index/introducing-upgrades-to-codex/), a new model optimized for agentic coding. The model was initially available within the Codex Cli, with support from [**GitHub**](https://github.blog/changelog/2025-09-23-openai-gpt-5-codex-is-rolling-out-in-public-preview-for-github-copilot/), [**Cursor**](https://x.com/cursor_ai/status/1970540811168473250), [**Windsurf**](https://x.com/windsurf/status/1970549712551100523) and [**Cline**](https://cline.bot/blog) arriving the following week. We should anticipate this model of releases becoming more frequent, with OpenAI using the model release to increase usage of the Codex CLI tool before making it generally available to other providers. We should also expect Anthropic to follow suit. 

* **OpenAI** introduced significant new features to the Codex CLI alongside the [GPT5-Codex](https://openai.com/index/introducing-upgrades-to-codex/) launch, including a new [code review capability](https://developers.openai.com/codex/cloud/code-review) and [an IDE extension](https://developers.openai.com/codex/ide). 

* **Google** [announced increased rate-limits](https://blog.google/technology/developers/gemini-cli-code-assist-higher-limits/) for Gemini CLI pro and enterprise users. 

* **Cursor** quietly shipped the cursor cli, which is also marketed as the cursor agent. It can be used as a CLI, in a headless mode and within GitHub Actions. The CLI is being positioned as part of their overall agentic AI offering. More details in the [docs](https://cursor.com/docs/cli/overview). Separately Cursor [shipped version 1.6](https://cursor.com/changelog), including a new summarise feature which summarises an existing chat session to reduce the number of tokens being used in a session. 

* **Windsurf** [announced a queued-messages feature](https://windsurf.com/blog/queued-messages), allowing chat users to submit “next action” messages to an llm rather than waiting for completion of the current task. 

* **Tessl**, an AI native developer tools company, founded by Guy Podjarny, previously the founding CEO of Snyk (see Mona Monitor [5](https://github.com/github/Business-Insights/discussions/10), [14](https://github.com/github/Business-Insights/discussions/21)), [released the first two components of their product offering](https://tessl.io/blog/announcing-tessls-products-to-unlock-the-power-of-agents/) - the tessl registry, which provides explicit guidance on how llms should use over 10,000 opensource libraries and components, and the tessl framework, a spec driven development framework. 

* **Factory AI,** an agentic AI tools vendor focused on running multiple parallel agents (sometimes referred to Swarm coding), [raised a $50M Series B](https://factory.ai/news/series-b), led by NEA and Sequoia. 

* **CodeRabbit**, an AI Code Review vendor, [announced a $60M series B](https://www.coderabbit.ai/blog/coderabbit-series-b-60-million-quality-gates-for-code-reviews), led by Scale Ventures. Coderabbit has raised $88M. Separately CodeRabbit [released a CLI](https://www.coderabbit.ai/blog/coderabbit-cli-free-ai-code-reviews-in-your-cli) in beta.

* **Greptile**, an AI code review vendor, [announced a $25M Series A](https://www.greptile.com/blog/series-a), led by Benchmark Capital. 

* **Vercel** [announced an AI Code Review Tool](https://vercel.com/changelog/ai-code-reviews-by-vercel-agent-now-in-beta), with specific optimizations for [next.js](http://next.js/) and other frameworks. Billing is usage based, and beta access is limited to Pro and Enterprise customers. 

* **Bito**, an AI code review vendor, [introduced a Jira integration](https://bito.ai/blog/jira-integration-what-shipped-this-week-09-18-25/) which is triggered on a pull request and pulls context from Jira in for the code review. 

* **Korbit**, an AI code review vendor, became the first vendor in the space [to publicly announce it was shutting down](https://www.korbit.ai/post/closure-announcement). 

* **Qodo**, an AI developer tools vendor, [introduced Qodo Aware](https://www.qodo.ai/blog/introducing-qodo-aware-deep-codebase-intelligence-for-enterprise-development/), billed as a deep intelligence tool for enterprise code bases. They have [provided a demo offering](https://github.com/qodo-ai/open-aware) which indexes 100 open source repos for customers to try.. Qodo [highlighted](https://www.qodo.ai/blog/qodo-named-a-visionary-in-the-2025-gartner-magic-quadrant-for-ai-code-assistants/) that they were named a visionary in the 2025 AICA MQ mentioned above. 

* **Replit**, a vibe coding platform, [announced a $250M funding round](https://replit.com/news/funding-announcement) led by Prysm Capital. Replit stated ARR has passed $150M in their funding press release. 

* **Lovable**, a vibe coding platform, [announced](https://lovable.dev/blog/lovable-file-uploads) a new file upload capability which automates conversion of certain types of resources into websites. 

* **Zed,** an AI enabled IDE, [changed their pricing models for LLMs](https://zed.dev/blog/pricing-change-llm-usage-is-now-token-based) to a purely usage based model. 

* **AugmentCode**, an AI Coding Assistant vendor,[ introduced a new $20 single developer tier,](https://www.augmentcode.com/blog/augment-is-now-more-affordable-introducing-our-usd20-per-month-indie-plan) limited to 125 messages. This marks a significant shift from their enterprise sales motion. They compare their offering to Claude Code Pro and Cursor Pro in their announcement post. 

* Crowdbotics, an AI modernization platform, rebranded to [**CoreStory**](https://corestory.ai/post/welcome-to-corestory). Modernization was a bit of a trend over the week with **Rocket Software** announcing an [AI Cobol Assistant](https://www.globenewswire.com/news-release/2025/09/16/3150734/0/en/Rocket-Software-Advances-COBOL-Modernization-with-GenAI-and-ARM-Ready-Deployment-Enabling-Disruption-Free-Transformation.html) and **Mongodb** announcing an [AI assisted application modernization tool](https://www.mongodb.com/press/mongodb-launches-ai-powered-application-modernization-platform-to-reduce-technical-debt-and-speed-innovation).

**DevOps**

* **Atlassian** [acquired **DX**](https://www.atlassian.com/blog/announcements/atlassian-acquires-dx), a software engineering intelligence vendor, for $1Bn. DX was founded by former hubber Abi Noda. Separately Jason Warner, CEO of **Poolside** and former CTO of **GitHub**, [joined Atlassian’s board](https://www.linkedin.com/posts/mcannonbrookes_big-welcome-to-jason-warner-as-he-joins-atlassian-activity-7374180132770480130-UigE/). 

* **Atlassian** [announced Rovo Chat for BitBucket.](https://www.atlassian.com/blog/bitbucket/rovo-chat-bitbucket-beta) It pulls context from Jira and Confluence into your overall BitBucket workflow. We should anticipate a GitHub targeted version of this coming in the near future. 

* **CircleCI** [announced Chunk](https://circleci.com/blog/introducing-chunk/), a new AI feature focused on "autonomous validation” of CI/CD pipelines. The underlying idea is that Chunk will fix failed builds, optimize configurations and identity broken tests. The offering is only available via wait list

* **GItLab** [released GitLab 18.4.](https://about.gitlab.com/blog/gitlab-18-4-ai-native-development-with-automation-and-insight/) Their announcements mainly focused on bringing AI into the SDLC.
